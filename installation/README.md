# installation/

Setup guides for local development dependencies (databases, runtimes,
tooling) needed to work on LeadPilot. Status: empty — populate once
the tech stack is chosen (see tech-stack/README.md).

## What belongs here

- Step-by-step install guides for anything needed before `commands/`
  will work (e.g. a database engine, a specific language runtime
  version manager, a CLI tool)
- One file per major dependency

## Suggested file naming

  (dependency-name).md, e.g. postgresql.md, python-environment.md

## Notes

- Keep this scoped to installation/setup only — day-to-day commands
  belong in commands/, not here.
